BACK TO RIJKS MIX

Visiting a classical art museum normally amounts to silent appraisal of exhibited art pieces. Silence indeed eliminates environmental distraction and allows the visitor to contemplate art, yet I would sometimes wish a classical art exhibition would not only stimulate my vision (and thought), but other senses as well. For my first Mashups project, I decided to create a multi sense art experience myself. Specifically, I used Rijskmuseum’s and Soundcloud’s API to juxtapose images of paintings and sound.

API'S

Rijksmuseum is a museum in Amsterdam dedicated to Dutch Art and history. The museum boasts a user-friendly API that allows developers to get information about art objects displayed in the museum. Importantly, one can also search for images of exhibited art pieces. Soundcloud is a major music distribution platform. Through its API, one can access music tracks as well as various user-related information.

RIJKS MIX

On the website that I created for the project, the user is supposed to type the name of his or her favorite painter into a search box. If the painter’s work is on display in Rijksmuseum, an image of the painter’s artwork appears on the screen. Simultaneously, the painter’s name is sent to Soundcloud API, which sends back a list of tracks corresponding to the search query. One of the tracks is then embedded into HTML’s audio element and played.

OBSTACLES

Whereas Rijskmuseum API documentation is easy to understand, the same does not hold true for Soundcloud’s documentation. The documentation focuses on Soundcloud’s SDK, which made it difficult for me to figure out how to request data via URL requests. In addition, Soundcloud API requires several forms of client and user authentication. It was only after considerable hassle that I understood how to incorporate Soundcloud’s tracks into my project.

FUTURE PLANS

Currently, juxtaposition of paintings and sound is the only thing taking place on the page. The project could benefit from some educational content, for instance the painter’s biography. Also, comments that Soundcloud users make about different tracks could be flashed on the screen. Furthermore, user preference algorithms could be used to track which paintings a particular user enjoys the most.
